## Description
#### Technologies

- [Nest](https://github.com/nestjs/nest)
-  Based on [Express](https://expressjs.com/)
- [Mongoose](https://mongoosejs.com/)
- [GraphQL](https://graphql.org/)
- [MongoDB](https://www.mongodb.com/)
- [Jwt](https://jwt.io/)


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
